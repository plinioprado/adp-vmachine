# Vending Machine Project

Vending machine, with no interface, that does:

* Print inventory
* Refill inventory
* Re-supply change
* Dispense inventory based on payment
* Return change as coins (eg. $0.35 is 1 quarter and 1 dime)


# Stack

Javascript ES6
Testing in Jest
